0. git clone https://github.com/johnson18404/SimpleDownloader.git && cd SimpleDownloader
1. Install: npm Install
2. Enter urls that you want to download.
3. Execute: npm start

- - -
詳細中文操作說明
1. 此程式使用Node.js語言撰寫，請至 https://nodejs.org/en/ 下在NodeJS安裝，LTS或Current版本都可以。
2. 點右上角綠色Clone or Download按鈕>Download Zip，下載檔案並解壓縮。
3. 開啟Terminal切換到解壓縮後目錄，輸入npm install安裝必要函示庫。
4. 在urls.txt輸入欲下載檔案的網址。
5. 在Terminal中輸入npm start即會開始下載。
